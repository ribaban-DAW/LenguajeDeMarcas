• C
    • C es un lenguaje de bajo nivel diseñado para ofrecer control directo sobre los recursos del hardware. Es eficiente y muy usado en programación de sistemas, sistemas operativos y aplicaciones embebidas. Ofrece gran control sobre la memoria mediante el uso de punteros.
  • Python
    • Python es conocido por su sintaxis simple y su enfoque en la legibilidad del código, lo que facilita el aprendizaje y la escritura de código limpio. Ofrece una gran cantidad de bibliotecas estándar y de terceros que lo hacen ideal para diversos tipos de proyectos.
  • Rust
    • Rust es un lenguaje moderno diseñado para ofrecer seguridad en la gestión de memoria sin sacrificar el rendimiento. Se enfoca en la concurrencia y la seguridad, evitando los problemas comunes de gestión de memoria que enfrentan otros lenguajes como C y C++.
  • C++
    • C++ es una extensión del lenguaje C que añade características de programación orientada a objetos, junto con soporte para programación genérica y metaprogramación. Mantiene las características de bajo nivel de C, lo que permite control directo sobre el hardware y gestión manual de memoria, pero también incluye abstracciones de alto nivel.
  • Java
    • Java es un lenguaje popular que se destaca por su capacidad de ser portable, gracias a la Máquina Virtual de Java (JVM). Es muy utilizado en aplicaciones empresariales debido a su robustez y capacidad de ejecución en múltiples plataformas sin cambios en el código.
